# mini-app

## About The Project

This mini application: 
* Has a Sign-Up Page with Option for Mentor and Mentee
* Has a Profile Page for Mentor
* Has a Profile Page for Mentee showing all the mentors assigned to the mentee
* Has a Database to store the Mentor and Mentee Data
* Uses PHP/CSS/HTML/JavaScript only no framework
* Uses Object Oriented Programme for the application
